## 🚀 Быстрый старт

### 1. Подготовка

```bash
# Клонировать репозиторий
git clone <repo-url>
cd pm-assistant

# Создать .env файл
cp env.example .env
```

### 2. Настроить .env

Обязательно настроить:
- `S3_ENDPOINT_URL`, `S3_ACCESS_KEY`, `S3_SECRET_KEY` - MinIO S3
- `OPENAI_API_KEY` - OpenAI ключ
- `AD_SERVER`, `AD_DOMAIN`, `AD_BASE_DN` - Active Directory

### 3. Запустить

```bash
# Запустить все сервисы
docker compose up -d

# Применить миграции
docker compose exec api alembic upgrade head

# Проверить логи
docker compose logs -f
```

### 4. Использовать

- API: http://localhost:8000
- Docs: http://localhost:8000/docs

## 📝 Примеры использования

### Создание проекта

```bash
curl -X POST "http://localhost:8000/api/projects/" \
  -H "Cookie: session_id=YOUR_SESSION" \
  -H "Content-Type: application/json" \
  -d '{
    "name": "Мой проект",
    "description": "Описание"
  }'
```

### Создание встречи с аудио

```bash
curl -X POST "http://localhost:8000/api/meetings/" \
  -H "Cookie: session_id=YOUR_SESSION" \
  -F "title=Встреча команды" \
  -F "project_id=1" \
  -F "audio_file=@meeting.mp3"
```

### Запуск обработки

```bash
curl -X POST "http://localhost:8000/api/meetings/1/process" \
  -H "Cookie: session_id=YOUR_SESSION"
```

### Проверка статуса

```bash
curl "http://localhost:8000/api/meetings/1/processing-status" \
  -H "Cookie: session_id=YOUR_SESSION"
```
